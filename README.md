# ScheduleMe-App

**ScheduleMe** is a field technician scheduling system that simplifies and streamlines weekly shift assignments across multiple regions. Built upon the foundation of my earlier **schedule-automation** project, this app has evolved into a full-featured tool tailored for operations teams managing technicians in the field.

It automatically assigns shifts based on seniority, preferences, and coverage rules — reducing manual scheduling headaches while ensuring fairness and operational efficiency.

## 🔧 Features

- Weekly schedule generation (Sunday to Saturday)
- Seniority-based shift assignments
- Honors shift and territory preferences with fallback logic
- Balanced technician coverage across many locations
- Minimum shift enforcement + weekend coverage requirement
- Retry logic for under-scheduled employees
- Region-level statistics dashboard for management oversight
- Fully interactive multi-page Streamlit interface
- Supports both sample data generation and real CSV uploads

## 📊 Built For

- Field operations teams managing machine maintenance across territories
- Managers who need clear oversight of shift coverage and employee load
